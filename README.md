# TasKing

## API

**Return all task:**

GET: `/api/tasks`

**Return task by id:**

GET: `/api/tasks/{task-id}`

**Return rankings of a task:**

GET: `/api/tasks/{task-id}/rankings`

**Return all task histories:**

GET: `/api/history/{task-id}`

**Save task:**

POST: `/api/tasks`

**Save history entry:**

POST: `/api/history`
